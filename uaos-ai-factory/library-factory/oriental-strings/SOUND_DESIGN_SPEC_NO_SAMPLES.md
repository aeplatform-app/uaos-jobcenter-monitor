# Sound Design Spec No Samples

Dieser Run enthält keine Audio-Samples. Sound Design wird nur als Spezifikation beschrieben.

Erlaubt:

- Synthese-Experimente.
- Eigene spätere Aufnahmen.
- Eigene MIDI- und Metadaten-Dateien.
- Layering-Konzepte ohne Fremdmaterial.

Verboten:

- Kopierte Samples.
- Kontakt- oder NI-Extraktion.
- Kommerzielle Loops.
- Nachbau geschützter Phrasen.
