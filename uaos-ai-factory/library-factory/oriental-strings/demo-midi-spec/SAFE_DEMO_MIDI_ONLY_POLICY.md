# Safe Demo MIDI Only Policy

Safe demo MIDI means original symbolic musical sketches only.

Allowed:

- JSON phrase plans.
- Markdown phrase descriptions.
- Later original `.mid` demo files only after explicit approval.

Blocked:

- Keyboard-native output.
- .STY/.SET/.PRS/.STL/.PAT/.MSP/.KST files.
- Copied melodies.
- Commercial loops.
- Proprietary library content.
