# UAOS Oriental Strings Product Vision

Datum: 01.07.2026

UAOS Oriental Strings soll eine erkennbare Signatur für Arabic, Levantine und Turkish expressive strings werden. Der Fokus liegt auf emotionaler Melodik, maqam-aware phrase behavior und sicherer Planung eigener Inhalte.

In diesem Run werden keine realen Samples erzeugt, kopiert oder extrahiert. Alle Inhalte sind Produktvision, Mapping, Metadaten, Demo-MIDI-Plan und Sound-Design-Spezifikation.

## Identität

- Arabic / Levantine / Turkish expressive strings.
- Emotionales Solo-Violin- und Ensemble-Gefühl.
- Maqam-aware Phrasenlogik.
- Quarter-tone planning als Metadaten, nicht als Keyboard-Ausgabe.
- Spätere originale Aufnahmen oder Synthese nur mit klarer Rechtekette.
