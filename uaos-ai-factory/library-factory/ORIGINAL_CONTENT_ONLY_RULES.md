# Original Content Only Rules

1. Keine externen Sample-Bibliotheken kopieren.
2. Keine Kontakt-, NI- oder kommerziellen Library-Dateien extrahieren.
3. Keine Loops nachbauen, wenn sie als Kopie erkennbar wären.
4. MIDI-Demos müssen eigene musikalische Ideen sein.
5. Sound-Design-Spezifikationen dürfen Referenzqualitäten beschreiben, aber keine geschützten Inhalte übernehmen.
6. Für Premium-Libraries werden später Aufnahmefreigaben, Performer-Releases und Metadaten benötigt.
7. Keyboard-native Ausgabe bleibt blockiert.
8. Payment bleibt blockiert.
