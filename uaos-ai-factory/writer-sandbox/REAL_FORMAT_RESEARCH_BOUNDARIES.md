# Real Format Research Boundaries

LOCAL ONLY - BOUNDARIES ONLY - NO REAL FORMAT OUTPUT

## Allowed

- Read public documentation at a high level in a future approved research task.
- Write local notes.
- Improve neutral JSON schemas.
- Improve validators for safety labels.
- Define test fixture policy without real keyboard-native output.

## Blocked

- No reverse engineering claims.
- No proprietary samples.
- No Kontakt or Native Instruments copying.
- No public release.
- No keyboard transfer.
- No `.SET` / `.STY` output.
- No fake real-format output.
- No binary legacy file content reads.

## Stop Conditions

- Any request to create real keyboard-native output.
- Any request to transfer to keyboard.
- Any request to use legacy `.STY` files.
- Any request to copy proprietary or sample-library content.
- Any request to claim compatibility before testing.

## Current Recommendation

Stay with neutral metadata validation and owner-neutral package revision until a separate high-risk gate is approved.
