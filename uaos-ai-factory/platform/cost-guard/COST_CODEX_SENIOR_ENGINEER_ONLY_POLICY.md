# Code X Senior Engineer Only Policy

Status: local plan only

Code X/Codex is reserved for senior engineering tasks that benefit from architectural judgment, complex debugging, safety review, or careful implementation.

It must not be used as an always-on worker, background agent, broad scanner, 24-hour agent, open loop executor, or cheap documentation worker.

Every Code X task must be small, bounded, reported, and stopped at the first serious FAIL.
