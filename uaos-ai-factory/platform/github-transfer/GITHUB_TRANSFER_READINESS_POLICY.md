# GitHub Transfer Readiness Policy

Status: local plan only

The repository transfer remains an owner-managed manual action. Local planning may document checks, risks, and sequence, but it must not perform a transfer, change the remote, publish commits, deploy, create payment flows, or enable writer/export behavior.

## Required Owner Checks

- Current origin is still `https://github.com/Sari-raslan/universal-arranger-os.git`.
- Planned future target is `https://github.com/aeplatform-app/universal-arranger-os.git`.
- Target repository availability is confirmed by the owner.
- AE Platform admin access is confirmed.
- `admin@aeplatform.app` is confirmed as the administrative contact.
- Branch protection plan is ready.
- GitHub Pages impact is understood.
- Secrets and environment variables are reviewed.
- Vercel integration is paused until staging rules are approved.
- `aeplatform.online` remains a future domain candidate until DNS is explicitly approved.
