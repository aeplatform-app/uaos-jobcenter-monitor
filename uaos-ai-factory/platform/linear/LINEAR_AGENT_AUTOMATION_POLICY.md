# Linear Agent Automation Policy

Future automation rules:
- Linear agent automation is planned only and inactive.
- Human approval is required before enabling any automation.
- AI can only work on Ready tasks.
- High-risk tasks require human approval.
- No task may trigger deploy, payment, real writer/export, DNS changes, or production release.
- GitHub branch/PR workflow is required before implementation work.
- Code X / Codex remains the senior engineer, not an always-on worker.
