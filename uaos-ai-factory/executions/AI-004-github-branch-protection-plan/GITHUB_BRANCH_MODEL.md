# GitHub Branch Model

- `master` / `main`: stable source of truth.
- `develop`: AI integration and testing.
- `staging`: Vercel preview/staging candidate.
- `ai/*`: Copilot, Code X, or OpenHands work branches.
- `docs/*`: planning and documentation.
- `chore/*`: maintenance.

The current repository uses `master`. Renaming `master` to `main` is a future decision only and is not part of this task.
