# Premium Library Schema Plan

Proposed metadata fields:
- `packId`
- `packName`
- `status`
- `sourceType`
- `sourceOwnership`
- `licenseStatus`
- `instrumentList`
- `categories`
- `articulations`
- `velocityLayers`
- `roundRobins`
- `sampleMapStatus`
- `qaStatus`
- `legalReviewStatus`
- `commercialClaimAllowed`
- `notes`

The schema is metadata-only and must not include audio paths or sample file references.
