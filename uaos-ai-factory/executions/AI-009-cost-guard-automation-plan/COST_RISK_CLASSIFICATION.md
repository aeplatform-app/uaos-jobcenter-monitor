# Cost Risk Classification

Status: local plan only

## LOW

Docs, planning, local metadata, reports, small templates, and narrow local checks.

Default route: local script or manual work.

## MEDIUM

Small UI/component changes, tests, and small local scripts.

Default route: Code X small bounded prompt or Copilot later, issue-bound and reviewed.

## HIGH

Architecture rewrites, build system changes, export/writer behavior, payment, deploy, real sample/audio work, and external APIs.

Default route: owner approval required before any work.

## BLOCKED

Production deploy, payment activation, real writer/export, proprietary content import, whole repo scan, destructive git history rewrite, and external automation without approval.

Default route: do not perform.
