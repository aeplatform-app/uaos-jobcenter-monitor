# Static Demo Gateway Safety Policy

LOCAL ONLY - NOT PUBLIC RELEASE

Status: local plan only

The static demo gateway is a local presentation aid. It may support screenshots, Jobcenter conversations, and supporter explanation, but it must not be described as a public release or production deployment.

## Required Limits

- Use locally only.
- Use for screenshots and demo presentation only.
- Do not run Vercel commands.
- Do not publish commits before GitHub transfer.
- Do not create payment flows.
- Do not claim real keyboard writer/export readiness.
- Do not import proprietary samples.
- Do not modify App.jsx for this handoff.
- Keep external automation blocked.
