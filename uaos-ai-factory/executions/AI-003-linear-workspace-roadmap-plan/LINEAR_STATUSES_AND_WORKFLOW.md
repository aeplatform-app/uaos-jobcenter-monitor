# Linear Statuses And Workflow

## Statuses

- Backlog
- Ready
- In Progress
- Review
- QA
- Blocked
- Waiting External Account
- Done Local
- Done
- Deferred

## Rules

- AI can only work on Ready tasks.
- External tasks remain blocked until account integration is manually approved.
- High-risk tasks need human approval.
- Real writer, payment, and deploy tasks are blocked by default.
