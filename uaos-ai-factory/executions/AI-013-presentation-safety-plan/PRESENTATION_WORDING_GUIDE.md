# Presentation Wording Guide

LOCAL ONLY - NOT PUBLIC RELEASE

Status: local plan only

## Safer Wording

- "Local prototype/demo material."
- "Static presentation gateway for screenshots."
- "Planning-only AI Factory safety work."
- "Funding-dependent next milestone."
- "No public release yet."
- "No production deployment."
- "No payment/live customer activity."
- "No real keyboard writer/export readiness."

## Avoid

- Launch wording.
- Production readiness wording.
- Payment readiness wording.
- Live customer wording.
- Real export readiness wording.
- Proprietary sample/library ownership wording.
- Any wording that implies external automation is already approved.
