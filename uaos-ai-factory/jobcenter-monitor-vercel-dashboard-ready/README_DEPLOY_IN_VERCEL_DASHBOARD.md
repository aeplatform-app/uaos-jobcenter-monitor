# UAOS Jobcenter Monitor - Vercel Dashboard Ready

This folder is a clean static package for a public UAOS / AE Platform Jobcenter monitor.

No deployment was attempted during package creation.

## Manual Dashboard Steps

1. Open Vercel Dashboard.
2. Use the existing account/team.
3. Create or select project name:
   - `aeplatform-private-demo-deploy-clean`
   - OR `uaos-jobcenter-monitor`
4. Import/connect this created static folder or the Git folder containing it.
5. Deploy as a static project with no build command.
6. Copy the Production URL after deploy.
7. Expected public routes:
   - `https://<vercel-project-url>/jobcenter/`
   - `https://<vercel-project-url>/status/`
8. After the public URL works, run the next task to update the Jobcenter PDF/PPTX with that URL.
