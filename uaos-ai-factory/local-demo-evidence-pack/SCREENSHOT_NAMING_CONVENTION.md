# Screenshot Naming Convention

LOCAL ONLY - PRIVATE REVIEW ONLY - NOT KEYBOARD OUTPUT

Recommended pattern:

`UAOS_LOCAL_<sequence>_<view>_<YYYYMMDD>.png`

Examples:

- `UAOS_LOCAL_01_DASHBOARD_TOP_20260701.png`
- `UAOS_LOCAL_02_SELECTED_NEUTRAL_PACKAGE_20260701.png`
- `UAOS_LOCAL_03_SONG_TO_ARRANGER_20260701.png`
- `UAOS_LOCAL_04_SEQUENCER_GRID_20260701.png`
- `UAOS_LOCAL_05_PIXI_ASSISTANT_20260701.png`
- `UAOS_LOCAL_06_SEND_READY_20260701.png`
- `UAOS_LOCAL_07_SAFE_EXPORT_CENTER_20260701.png`
- `UAOS_LOCAL_08_EVIDENCE_PACK_FOLDER_20260701.png`

Rules:
- Use local screenshots only.
- Do not include customer names, passwords, tokens, or private documents.
- Do not name anything as public, deployed, paid, or keyboard output.
