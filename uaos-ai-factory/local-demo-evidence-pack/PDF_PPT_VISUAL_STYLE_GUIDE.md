# PDF/PPT Visual Style Guide

LOCAL ONLY - STYLE PLANNING ONLY - NO PDF/PPTX CREATED IN THIS RUN

## Tone

- Clear, calm, factual.
- Private review framing.
- Safety boundaries visible on every major section.

## Layout

- Use short headings.
- Use one main idea per page or slide.
- Prefer tables for checks, status, and decisions.
- Keep screenshots large enough to read.

## Visual Rules

- Use simple neutral backgrounds.
- Avoid public-launch, payment, deployment, or keyboard-output language.
- Add a local-only footer where useful.
- Label screenshots as local private demo evidence.

## Required Footer Options

- LOCAL ONLY - PRIVATE REVIEW ONLY
- NOT KEYBOARD OUTPUT
- NO PUSH / NO DEPLOY / NO VERCEL
