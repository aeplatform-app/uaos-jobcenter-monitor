# UAOS AI Factory

UAOS AI Factory is a controlled framework for future AI-assisted development on Universal Arranger OS. It is prototype, demo, and staging oriented. It does not start autonomous background work, deploy software, accept payments, publish releases, or enable real keyboard writing/export.

Every worker must operate from a tracked issue, use a branch and pull request, respect cost controls, and stop at the first serious failure. Code X / Codex acts as the senior engineer and safety reviewer, not as an always-running worker.

Real keyboard writer/export features, public release, production release, and payment flows require explicit human approval before any implementation or activation.
