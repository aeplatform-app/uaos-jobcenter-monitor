# UAOS Bounded Local Agent Policy

Status: LOCAL ONLY

This policy allows only small local work packets after the AI Factory final freeze.

## Rules

- No 24-hour agents.
- No open-ended agents.
- One small task at a time.
- Stop after a written report.
- Commit local only after requested checks pass.
- Owner approval is required for risky tasks.
- No proprietary assets.
- No Kontakt, Native Instruments, sample library, or copied sample content.
- No real keyboard writer/export.
- No App.jsx changes.
- No production release.
- No external automation.
- No background agents.
- No hosting publication.
- No paid-flow creation.

## Routing

- LOW risk: local script or manual work only.
- MEDIUM risk: Code X small bounded prompt only.
- HIGH risk: owner approval required before work.
- BLOCKED risk: do not perform.

## Stop Rule

If scope, ownership, risk, or allowed files are unclear, stop and write a local blocker note instead of continuing.
