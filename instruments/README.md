# UAOS Instruments

Goal:
Build our own sampler and instrument engine inspired by professional sample players.

We do NOT copy Kontakt or Native Instruments sounds.
We build a legal original UAOS instrument system.

## Focus
- Piano engine
- Sampler engine
- Presets
- Velocity layers
- Round robins
- ADSR envelope
- Reverb
- EQ
- MIDI input
- Export ready sounds
