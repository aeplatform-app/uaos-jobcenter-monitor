# UAOS DevOps Tools

Start all local services:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\devops\uaos-launcher.ps1
```

Check service health:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\devops\uaos-health.ps1
```

Stop local services:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\devops\uaos-stop.ps1
```
