# UAOS Studio Product Vision

UAOS Studio should become a modern AI music workstation.

Instead of copying old DAWs, it focuses on:
- faster idea capture
- AI arrangement
- voice to MIDI
- chord intelligence
- smart orchestration
- automatic demo generation
- export to other DAWs
- creator-friendly workflows

## Difference from traditional DAWs
Traditional DAWs start with tracks.
UAOS starts with the musical idea.

Input:
- voice
- humming
- MIDI
- audio
- text prompt

Output:
- arrangement
- MIDI
- chords
- stems
- song structure
- mix concept
