# UAOS Studio

Goal:
Build a new AI-powered music workstation inspired by professional DAWs, without copying proprietary software.

We do not clone Cubase.
We learn from general DAW concepts and create a new system.

## Core Ideas
- Timeline editor
- MIDI editor
- Audio tracks
- Chord track
- Arrangement assistant
- Voice to MIDI
- AI orchestration
- DAW export
- Plugin-ready architecture

## Supported analysis formats
- MIDI
- WAV
- MP3
- MusicXML
- stems
- exported audio

## Not allowed
- No copying Cubase UI
- No decompiling Cubase
- No cracking CPR project files
- No bypassing licenses
