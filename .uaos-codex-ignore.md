# UAOS Codex Ignore Guidance

Use these paths and patterns as default exclusions for fast, targeted Code X work.

- `node_modules/`
- `.git/`
- `uaos-live-clean/dist/`
- `backups/`
- `**/pdf-output/`
- `**/ppt-output/`
- `**/*.zip`
- `**/*.pdf`
- `**/*.pptx`
- `**/*.png`
- `**/*.jpg`
- `**/*.mp4`
- `**/*.wav`
- `**/*.mp3`

Do not inspect these files or folders unless the user explicitly requests it or the task requires a targeted existence/size check.
