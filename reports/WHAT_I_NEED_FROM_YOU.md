# UAOS REQUIRED FROM SAREY

## 1\. Backend production

اختار واحد:

* Render
* Railway
* Fly.io

المطلوب منك:

* إنشاء حساب
* ربط GitHub repo
* اختيار مجلد backend
* وضع PORT=8080

## 2\. AI real engine

المطلوب منك:

* OpenAI API key أو موديل محلي
* تحديد هل AI يكون:

  * song analyzer
  * style generator
  * chord assistant
  * melody assistant

## 3\. MIDI hardware

المطلوب منك:

* توصيل كيبورد MIDI بالـ USB
* تحديد الجهاز الأساسي:

  * KORG PA3X
  * KORG PA5X
  * Yamaha Genos
  * Roland BK9
  * Ketron SD9

## 4\. Audio engine

المطلوب منك:

* هل تريد sampler داخلي؟
* هل تريد WAV export؟
* هل تريد drum engine؟
* هل تريد VST لاحقًا؟

## 5\. Export الحقيقي

اختار الأولوية:

1. KORG
2. Yamaha
3. Roland
4. Ketron

## 6\. Android/Desktop

المطلوب منك:

* هل تريد أولًا Desktop Windows؟
* أو Android APK؟
* لو Android: تحتاج signing key لاحقًا

## 7\. اسم المنتج

اختار الاسم النهائي:

* UAOS
* UA HyperStation
* Universal Arranger OS
* اسم آخر
